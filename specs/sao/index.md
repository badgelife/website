# SAOv1 Shitty Add-On Specification
The original SAO spec was creatd by @bbenchoff of Hackaday and more via the #Badgelife Slack, and to date has been maintained on [Hackaday](https://hackaday.io/project/52950-shitty-add-ons).  
  
However, modern SAOs currently utilize the [SAO 1.69bis spec](https://hackaday.com/2019/03/20/introducing-the-shitty-add-on-v1-69bis-standard/), aka SAOv2, which is documented in [several places](https://badge.life/specs/sao-v2), some with different information than others. Those are documented here.
